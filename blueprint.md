# Lotto Number Generator

## Overview
This is a simple web application that generates and displays a set of 6 unique lottery numbers, typically for a game like the Korean Lotto 6/45.

## Features
- Generates 6 unique random numbers between 1 and 45.
- Displays the numbers in a visually appealing way.
- A button to generate a new set of numbers.
- Responsive design that works on mobile and desktop.

## Design
- **Layout:** A clean, centered layout using modern CSS (Flexbox).
- **Color Palette:** A vibrant and energetic color palette.
- **Typography:** Clear and readable fonts with emphasis on the generated numbers.
- **Effects:** Subtle shadows and gradients for a modern look.
